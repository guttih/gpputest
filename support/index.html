GppUTest RPM package

Contains scripts to help with creating a new c++ project which includes unit tests allowing you to develop using a Test Driven Development (TDD).

To be able to use these scripts cpputest needs to be installed.

Some reading material

  • https://cpputest.github.io
  • Package README.md

Scripts in package

  • gpputest-install.sh
  • gpputest-newQtProject.sh
  • gpputest-setupTest.sh

Installing package from guttih repository

  1. Install the guttih repository 
    echo -e "[guttih]\nname=guttih repository\nbaseurl=https://guttih.com/public/vault/repo/guttih\ngpgcheck=0\nenabled=1" | sudo tee "/etc/yum.repos.d/guttih.repo"
    • Use dnf to install the package from guttih repository 
      dnf -y install gpputest
    • Use dnf to un-install the package 
      dnf -y remove gpputest

Manual install

  1. Install the package 
    wget https://guttih.com/public/vault/repo/guttih/gpputest-1.0.0-1.el8.noarch.rpm
sudo rpm --install gpputest-1.0.0-1.el8.noarch.rpm
  2. Un-install the package 
    sudo rpm --erase gpputest

Get tar file

wget https://guttih.com/public/vault/repo/assets/release/gpputest-1.0.0.tar.gz

Example

How to create a new QT project for TDD

mkdir -p ~/newProject 
cd ~/newProject
gpputest-newQtProject.sh -appname helloWorld
gpputest-setupTest.sh  -dir ~/newProject  -appdir ~/newProject/src  -codedir ~/newProject/src/code

And run all tests and show test coverage

make testcov
