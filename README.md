# gpputest
Scripts to help with creating TDD c++ projects with CppUTest

**gpputest** is a installable rpm for Red Hat-Based Linux like
  - fedora
  - Centos
  - Rocky linux
  
The package has been tested on *Rocky Linux 8.6 (Green Obsidian)*

## RPMs
All RPM packages are stored in the rpm directory.

A **test.rpm** test file can be download from this path
- [https://raw.githubusercontent.com/guttih/gpputest/main/rpm/test.rpm](https://raw.githubusercontent.com/guttih/gpputest/main/rpm/test.rpm)
